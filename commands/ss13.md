# SS13 - MANUS GIT AUTO-SYNC

Checkpoint + Push.

## MISSION
Create a checkpoint and sync to GitHub in one command.

## EXECUTION

First, create a checkpoint of the current workspace in Manus named: "Pre Git Sync Checkpoint - {current project name}".

After the checkpoint is created, run the following as a single non-interactive shell command from the project root:

```bash
[ -d .git ] || git init; \
REMOTE_URL="$(git remote get-url origin 2>/dev/null || echo "")"; \
if [ -n "$REMOTE_URL" ]; then \
  git add . && git commit -m "Instant Sync from Manus" || true; \
  git push -f origin "$(git symbolic-ref --short HEAD 2>/dev/null || echo main)" || \
  git push -f origin HEAD:main; \
else \
  REPO_SLUG="$(basename "$(pwd)" | tr ' ' '-' | tr '[:upper:]' '[:lower:]')"; \
  REPO_NAME="2025-manus-${REPO_SLUG}"; \
  git add .; \
  git commit -m "First Commit" || true; \
  git branch -M main 2>/dev/null || git checkout -b main; \
  gh repo create "$REPO_NAME" --private --source=. --remote=origin --push; \
fi; \
REMOTE_URL="$(git remote get-url origin 2>/dev/null || echo "")"; \
echo "STATUS: This Project is synced with $REMOTE_URL"
```

**PROJECT**: $ARGUMENTS

EXECUTE NOW.
