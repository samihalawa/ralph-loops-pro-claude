# SS18 - MEETING INTELLIGENCE & SPEC EXTRACTOR

Turn talk into action.

## MISSION
Extract a rigorous technical/business spec from unstructured conversation (transcripts/notes).

## NON-NEGOTIABLES
- Quote the source (Forensic trace)
- Separate "Wants" vs. "Needs"
- Identify the "Economic Buyer" vs. "User"

## PHASE 1 — CONTEXT MINING
- Scan for: Decisions, Action Items, Deadlines, Constraints
- Identify "Pain Points" (High emotion/friction)

## PHASE 2 — SYNTHESIS
- Draft the "Single Source of Truth" spec
- Define Acceptance Criteria
- List Open Questions (Risks)

## PHASE 3 — ACTION PLAN
- Immediate next steps (24h)
- Long-term roadmap

## OUTPUT CONTRACT
1. **Executive Summary**
2. **Requirements Spec** (Table)
   - ID | Requirement | Priority | Owner | Deadline
3. **Action Items** (Owner + Deadline)
4. **Open Questions** (Risks)

**TRANSCRIPT/NOTES**: $ARGUMENTS

EXECUTE NOW.
